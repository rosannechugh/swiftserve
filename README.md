# SwiftServe – Food Delivery Management System

A JavaFX-based desktop application that simulates a real-time food delivery system.

## ✨ Features
- 📦 Order creation and tracking
- 🚚 Delivery agent assignment
- 🔄 Real-time order lifecycle simulation
- 💰 Revenue tracking dashboard
- 🔍 Order filtering (Pending / Delivered)
- 🌙 Dark-themed UI

## 🛠️ Tech Stack
- Java
- JavaFX

## ▶️ How to Run
1. Download JavaFX SDK
2. Add VM options:
--module-path "path_to_javafx/lib" --add-modules javafx.controls,javafx.fxml

3. Run Main.java
